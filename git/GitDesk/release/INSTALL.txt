📦 INSTALLATION
═══════════════════════════════════════════════════════════════════

1. REQUIREMENT: .NET 9.0 Runtime
   └─ Download: https://dotnet.microsoft.com/download/dotnet/9.0
   └─ Or run: winget install Microsoft.DotNet.Runtime.9

2. RUN GitDesk.exe
   └─ No installation needed
   └─ Portable — run from any location
   └─ Total size: ~2.5 MB

📖 QUICK START
═══════════════════════════════════════════════════════════════════

Open a repo:        Ctrl+O
Clone a repo:       Click "📋 Clone"
Stage all:          Ctrl+Shift+A
Commit:             Ctrl+Enter
Push:               Ctrl+Shift+U
Show shortcuts:     Ctrl+/
Refresh:            F5

Full guide: See QUICKSTART.md

🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

Q: "Application won't start"
A: Install .NET 9.0 Runtime (see above)

Q: "Not a valid repository"
A: Ensure folder contains .git directory

Q: "Fetch/Pull/Push failed"
A: Check internet connection and Git credentials

Q: "Changes not showing"
A: Press F5 to refresh
